ProyectoMosquito
Este Proyecto consiste en la detección y conteo de larvas. Para correrlo se requiere:
Hardware:
1.Jetson Nano con cámara CSI.
2.Raspberry pi 2 con arducam y conexion a internet.
3.ESP32-cam.
Software:
1.Jetpack 4.5.1
2.mqtt-paho, python3, configuración del cron.
3.Arduino IDE configurado para ESP32: https://randomnerdtutorials.com/installing-the-esp32-board-in-arduino-ide-mac-and-linux-instructions/
  Ademas instalar las Librerias: PubSubClient, ArduinoJson.
